## Agregar
**Linting y formato**:
Ejecutar los scripts para automatizar tareas de linting.

```bash
npm run lint
```
```bash
npm run lint:fix
```
Ejecutar los scripts para garantizar un formato de código consistente.
```bash
npm run format
```
```bash
npm run format:check
```

**Mejoras de SEO**:
  - Se agregó un componente SEO a las páginas MDX para incluir dinámicamente metadatos de `palabras clave` y `última modificación` para una mejor generación de SEO.

## Actualización
**Actualización de Next.js**:
  - Se actualizó el proyecto de **Next.js 14** a **Next.js 15**, aprovechando las últimas características y mejoras de rendimiento.

**Mejoras en la base de código**:
  - Se reordenaron las importaciones en los archivos del proyecto para lograr una mejor consistencia y legibilidad.

**Componente Mermaid**:
  - Se corrigió un error de identificación que causaba problemas de representación cuando se incluían varios diagramas Mermaid en páginas MDX.
  - Se mejoró la lógica de inicialización y renderizado para evitar la representación duplicada de diagramas.
