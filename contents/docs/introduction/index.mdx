## Documents

Bienvenido a **Documents**, un kit de inicio de documentación de Next.js potente y flexible diseñado para simplificar el proceso de creación de documentación de productos, manuales técnicos y guías comerciales
de alta calidad y completas. Creado con **React**, **Tailwind CSS** y **TypeScript**, este kit de inicio proporciona la base
para crear la documentación de su proyecto y su base de conocimientos.

Documents se diseñó para agilizar la creación, la gestión y la presentación de la documentación, lo que permite a los desarrolladores y a los equipos centrarse en el contenido en lugar de
crear la estructura subyacente. Esta documentación tiene como objetivo ayudarlo a comenzar a utilizar el kit, explorar las distintas funciones y brindarle
información sobre cómo estructurar y presentar su contenido de manera eficaz.

## ¿Por qué documentos?

Documents se creó como un proyecto de código abierto para ayudar a la comunidad a optimizar su proceso de documentación, lo que permite a los desarrolladores centrarse en tareas más importantes. Se creó teniendo en cuenta la escalabilidad y se adapta a proyectos de todos los tamaños, desde guías de una sola página hasta bases de conocimiento completas.

La integración de MDX, componentes personalizados y Tailwind CSS garantiza una experiencia intuitiva, simplificando tanto la creación como la gestión de contenidos.

- **Guías de productos**: Manuales completos de uso, configuración y resolución de problemas del producto.
- **Documentación comercial**: Documentación clara y estructurada de procesos, políticas y sistemas internos.
- **Manuales Técnicos**: Guías detalladas para API, kits de desarrollo de software (SDK) y procesos técnicos.

Documents ofrece amplias opciones de personalización, lo que le permite alinear fácilmente el diseño y la funcionalidad con su marca.

## Características clave

Estas son algunas de las características que hacen que **Documents** sea ideal para crear su documentación:

| Característica                  | Descripción                                                                            |
| ------------------------------- | -------------------------------------------------------------------------------------- |
| **Creación de contenido**       | Escriba documentación utilizando Markdown con JSX, componentes personalizados y Mermaid.js para diagramas. |
| **Navegación y estructura**     | Organice el contenido con navegación de múltiples niveles, tablas de contenido generadas automáticamente y paginación fluida. |
| **Características del código**  | Resaltado de sintaxis, bloques de código mejorados y un conmutador de código con funcionalidad de copia.  |
| **Búsqueda avanzada**           | Búsqueda difusa con resaltado de términos para obtener resultados rápidos y precisos.       |
| **Tematización y diseño**       | Admite modos claro/oscuro con cambio automático y diseño adaptable.                |
| **Matemáticas y tablas**        | Integración perfecta de LaTeX para matemáticas y tablas estructuradas.        |
| **SEO y metaetiquetas**         | SEO integrado, generación de metadatos y control sobre metaetiquetas para mejores clasificaciones de búsqueda y compartición social.  |
| **Documentos basados ​​en IA (en desarrollo)** | Futura función para debates y búsquedas de contenido impulsados ​​por IA.         |

## Cómo utilizar estos documentos

Encontrará la barra de navegación de la documentación en el lado izquierdo de la pantalla. Las páginas están organizadas en una secuencia lógica, que va desde los temas básicos hasta los conceptos más avanzados, lo que le permite seguir paso a paso la creación de su documentación. Sin embargo, siéntase libre de explorar el contenido en cualquier orden y seleccionar las secciones más relevantes para su caso de uso.

En el lado derecho de la pantalla, una tabla de contenidos permite navegar fácilmente entre las secciones de cada página. Para acceder rápidamente a un contenido específico, puede utilizar la barra de búsqueda en la parte superior o el acceso directo de búsqueda (`Ctrl+K` o `Cmd+K`).

Dirígete al [Guía de instalación](/docs/introduction/installation).
