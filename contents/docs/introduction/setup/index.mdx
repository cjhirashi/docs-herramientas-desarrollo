Configurar su nuevo proyecto de documentación es muy sencillo. La mayoría de los ajustes esenciales del proyecto se pueden configurar dentro de la carpeta `settings`.

## Ajustes

Esta sección proporciona las configuraciones principales para su sitio de documentación, como definir la URL del sitio, el ícono y el título del sitio.

```tsx showLineNumbers
export const url = ""                 // La URL de su sitio de documentación
export const siteicon = ""            // Icono que se muestra junto al nombre del sitio en el encabezado
export const sitename = ""            // Título de su sitio de documentación
```

## Etiquetas de Google

Esta sección te permite integrar Google Tag Manager en tu proyecto de documentación. Puedes agregar tu código GTM y configurar si está conectado.

```tsx showLineNumbers
export const gtm = ""                 // Añade tu código de Google Tag Manager aquí
export const gtmconnected = true      // Establezca en verdadero si GTM está conectado; de lo contrario, configúrelo en falso
```

## Motor de búsqueda

Configure los ajustes esenciales de SEO para garantizar que su sitio de documentación esté optimizado para los motores de búsqueda. Esto incluye metadescripciones, palabras clave y detalles para compartir en redes sociales, como imágenes y texto alternativo.

```tsx showLineNumbers
export const description = ""         // Descripción de su sitio de documentación para SEO
export const keywords = ["", ""]      // Lista de palabras clave SEO para su sitio de documentación
export const urlimage = ""            // La URL de la imagen utilizada al compartirla en las redes sociales.
export const imagealt = ""            // Texto alternativo para imágenes compartidas, útil para lectores de pantalla
export const twitterhandle = ""       // El nombre de usuario de Twitter de su empresa para compartir en redes sociales
```

## Marca de pie de página

La configuración del pie de página le permite personalizar la información de la empresa que se muestra en la parte inferior del sitio. Puede agregar el nombre de la empresa y la URL a la que se vinculará el nombre.

```tsx showLineNumbers
export const companyname = ""         // El nombre de la empresa que aparece en la sección de derechos de autor
export const companylink = ""         // La URL a la que debe vincularse el nombre de su empresa
```

## Diseño general

Las configuraciones de diseño controlan la apariencia y funcionalidad de varios elementos del sitio de documentación, como la marca, la barra lateral derecha, las opciones de comentarios, la tabla de contenido y la funcionalidad de desplazamiento hacia la parte superior.

```tsx showLineNumbers
export const branding = true          // Establezca como verdadero para mostrar nuestros elementos de marca
export const rightsidebar = true      // Establezca como verdadero para habilitar la barra lateral derecha con comentarios, tabla de contenido y desplazamiento hacia arriba
export const feedbackedit = true      // Control para habilitar o deshabilitar la retroalimentación en GitHub
export const tableofcontent = true    // Control para habilitar/deshabilitar la tabla de contenidos generada
export const totopscroll = true       // Control para habilitar/deshabilitar el botón de desplazamiento hacia arriba
```

## Control de Github

Si sus archivos MDX están alojados en GitHub, puede configurar el proyecto para cargar documentos directamente desde allí. Esta opción puede resultar útil para sincronizar las actualizaciones de la documentación directamente con su repositorio.

```tsx showLineNumbers
export const loadfromgithub = false   // Establezca como verdadero para cargar sus documentos MDX directamente desde GitHub
```
