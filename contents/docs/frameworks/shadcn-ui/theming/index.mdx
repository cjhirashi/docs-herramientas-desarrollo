Puede elegir entre utilizar variables **CSS** (recomendado) o clases de utilidad para la creación de temas.

## VARIABLES CSS

```tsx
<div className="bg-background text-foreground" />
```

Para usar variables **CSS** para la creación de temas, configure `tailwind.cssVariables` como verdadero en su archivo `component.json`.

`component.json`
```json
{
    "style": "default",
    "rsc": true,
    "tailwind": {
        "config": "",
        "css": "app/globals.css",
        "baseColor": "neutral",
        "cssVariables": true
    },
    "aliases": {
        "components": "@/components",
        "utils": "@/lib/utils",
        "ui": "@/registry/new-york-v4/ui",
        "lib": "@/lib",
        "hooks": "@/hooks"
    },
    "iconLibrary": "lucide"
}

```

## CLASES DE UTILIDAD 

```tsx
<div className="bg-zinc-950 dark:bg-white" />
```

Para usar clases de utilidad para la creación de temas, configure `tailwind.cssVariables` como `false` en su archivo `component.json`.

`component.json`
```json
    {
    "style": "default",
    "rsc": true,
    "tailwind": {
        "config": "",
        "css": "app/globals.css",
        "baseColor": "slate",
        "cssVariables": false
    },
    "aliases": {
        "components": "@/components",
        "utils": "@/lib/utils",
        "ui": "@/registry/new-york-v4/ui",
        "lib": "@/lib",
        "hooks": "@/hooks"
    },
    "iconLibrary": "lucide"
}

```

## CONVENCION 

Utilizamos una convención simple de fondo y primer plano para los colores. La variable de fondo se utiliza para el color de fondo del componente y la variable de primer plano se utiliza para el color del texto.

<Note >
    El sufijo de fondo se omite cuando la variable se utiliza para el color de fondo del componente.
</Note>

Dadas las siguientes variables CSS:

```css
--primary: oklch(0.205 0 0);
--primary-foreground: oklch(0.985 0 0);
```

El color de fondo del siguiente componente será `var(--primary)` y el color de primer plano será `var(--primary-foreground)`.

```tsx
<div className="bg-primary text-primary-foreground">Hello</div>
```

## COMO AGREGAR NUEVOS COLORES 

Para agregar nuevos colores, debes agregarlos a tu archivo CSS y a tu archivo `tailwind.config.js`.

`app/globals.css`
```css
:root {
    --warning: oklch(0.84 0.16 84);
    --warning-foreground: oklch(0.28 0.07 46);
}

.dark {
    --warning: oklch(0.41 0.11 46);
    --warning-foreground: oklch(0.99 0.02 95);
}

@theme inline {
    --color-warning: var(--warning);
    --color-warning-foreground: var(--warning-foreground);
}
```

Ahora puede utilizar la clase de utilidad de advertencia en sus componentes.

```tsx
<div className="bg-warning text-warning-foreground" />
```
