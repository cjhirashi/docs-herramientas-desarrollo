Las cadenas son uno de los tipos de datos más fundamentales en Python, utilizadas para representar texto. Python ofrece una rica variedad de métodos integrados para manipular y trabajar con cadenas de manera eficiente. Este documento explora en detalle estos métodos, proporcionando ejemplos y casos de uso para cada uno.

### OBJETIVOS CLAVE

1.  **Manipulación de Cadenas:** Aprender a modificar cadenas, como cambiar mayúsculas/minúsculas, eliminar espacios en blanco y dividir/unir cadenas.
2.  **Búsqueda y Reemplazo:** Dominar las técnicas para buscar subcadenas dentro de una cadena y reemplazarlas.
3.  **Validación de Cadenas:** Utilizar métodos para verificar si una cadena cumple con ciertos criterios, como ser numérica, alfabética o contener solo espacios en blanco.
4.  **Formateo de Cadenas:** Explorar las diferentes formas de formatear cadenas para insertar valores y crear salidas personalizadas.

### METODOS DE MANIPULACION DE CADENAS

*   **`lower()`:** Convierte todos los caracteres de una cadena a minúsculas.

    ```python
    texto = "Hola Mundo"
    texto_minusculas = texto.lower()
    print(texto_minusculas)  # Output: hola mundo
    ```
*   **`upper()`:** Convierte todos los caracteres de una cadena a mayúsculas.

    ```python
    texto = "Hola Mundo"
    texto_mayusculas = texto.upper()
    print(texto_mayusculas)  # Output: HOLA MUNDO
    ```
*   **`capitalize()`:** Convierte el primer carácter de una cadena a mayúscula y el resto a minúsculas.

    ```python
    texto = "hola mundo"
    texto_capitalizado = texto.capitalize()
    print(texto_capitalizado)  # Output: Hola mundo
    ```
*   **`title()`:** Convierte el primer carácter de cada palabra en una cadena a mayúscula.

    ```python
    texto = "hola mundo"
    texto_titulo = texto.title()
    print(texto_titulo)  # Output: Hola Mundo
    ```
*   **`swapcase()`:** Invierte las mayúsculas y minúsculas en una cadena.

    ```python
    texto = "Hola Mundo"
    texto_invertido = texto.swapcase()
    print(texto_invertido)  # Output: hOLA mUNDO
    ```
*   **`strip()`:** Elimina los espacios en blanco al principio y al final de una cadena.

    ```python
    texto = "   Hola Mundo   "
    texto_sin_espacios = texto.strip()
    print(texto_sin_espacios)  # Output: Hola Mundo
    ```
*   **`lstrip()`:** Elimina los espacios en blanco al principio de una cadena.

    ```python
    texto = "   Hola Mundo   "
    texto_sin_espacios_izquierda = texto.lstrip()
    print(texto_sin_espacios_izquierda)  # Output: Hola Mundo
    ```
*   **`rstrip()`:** Elimina los espacios en blanco al final de una cadena.

    ```python
    texto = "   Hola Mundo   "
    texto_sin_espacios_derecha = texto.rstrip()
    print(texto_sin_espacios_derecha)  # Output:   Hola Mundo
    ```
*   **`removeprefix(prefix)` (Python 3.9+):** Elimina un prefijo específico de una cadena si está presente.

    ```python
    texto = "PrefijoHola Mundo"
    texto_sin_prefijo = texto.removeprefix("Prefijo")
    print(texto_sin_prefijo)  # Output: Hola Mundo
    ```
*   **`removesuffix(suffix)` (Python 3.9+):** Elimina un sufijo específico de una cadena si está presente.

    ```python
    texto = "Hola MundoSufijo"
    texto_sin_sufijo = texto.removesuffix("Sufijo")
    print(texto_sin_sufijo)  # Output: Hola Mundo
    ```

### METODOS DE BUSQUEDA Y REEMPLAZO

*   **`find(subcadena, inicio, fin)`:** Busca la primera ocurrencia de una subcadena dentro de una cadena y devuelve su índice. Devuelve -1 si no se encuentra. Los argumentos `inicio` y `fin` son opcionales y especifican el rango de búsqueda.

    ```python
    texto = "Hola Mundo, Hola Python"
    indice = texto.find("Hola")
    print(indice)  # Output: 0
    indice2 = texto.find("Hola", 1)
    print(indice2) # Output: 12
    ```
*   **`rfind(subcadena, inicio, fin)`:** Busca la última ocurrencia de una subcadena dentro de una cadena y devuelve su índice. Devuelve -1 si no se encuentra.

    ```python
    texto = "Hola Mundo, Hola Python"
    indice = texto.rfind("Hola")
    print(indice)  # Output: 12
    ```
*   **`index(subcadena, inicio, fin)`:** Similar a `find()`, pero lanza una excepción `ValueError` si la subcadena no se encuentra.

    ```python
    texto = "Hola Mundo"
    indice = texto.index("Mundo")
    print(indice)  # Output: 5
    ```
*   **`rindex(subcadena, inicio, fin)`:** Similar a `rfind()`, pero lanza una excepción `ValueError` si la subcadena no se encuentra.

    ```python
    texto = "Hola Mundo, Hola Python"
    indice = texto.rindex("Hola")
    print(indice)  # Output: 12
    ```
*   **`startswith(prefijo, inicio, fin)`:** Verifica si una cadena comienza con un prefijo específico.

    ```python
    texto = "Hola Mundo"
    comienza_con_hola = texto.startswith("Hola")
    print(comienza_con_hola)  # Output: True
    ```
*   **`endswith(sufijo, inicio, fin)`:** Verifica si una cadena termina con un sufijo específico.

    ```python
    texto = "Hola Mundo"
    termina_con_mundo = texto.endswith("Mundo")
    print(termina_con_mundo)  # Output: True
    ```
*   **`replace(viejo, nuevo, conteo)`:** Reemplaza todas las ocurrencias de una subcadena con otra. El argumento `conteo` es opcional y especifica el número máximo de reemplazos a realizar.

    ```python
    texto = "Hola Mundo, Hola Python"
    nuevo_texto = texto.replace("Hola", "Adiós")
    print(nuevo_texto)  # Output: Adiós Mundo, Adiós Python
    ```
*   **`count(subcadena, inicio, fin)`:** Cuenta el número de ocurrencias de una subcadena dentro de una cadena.

    ```python
    texto = "Hola Mundo, Hola Python"
    conteo = texto.count("Hola")
    print(conteo)  # Output: 2
    ```

### METODOS DE VALIDACION DE CADENAS

*   **`isalnum()`:** Devuelve `True` si todos los caracteres de la cadena son alfanuméricos (letras o números) y la cadena no está vacía.

    ```python
    texto1 = "HolaMundo123"
    texto2 = "Hola Mundo"
    print(texto1.isalnum())  # Output: True
    print(texto2.isalnum())  # Output: False
    ```
*   **`isalpha()`:** Devuelve `True` si todos los caracteres de la cadena son letras y la cadena no está vacía.

    ```python
    texto1 = "HolaMundo"
    texto2 = "Hola Mundo123"
    print(texto1.isalpha())  # Output: True
    print(texto2.isalpha())  # Output: False
    ```
*   **`isdigit()`:** Devuelve `True` si todos los caracteres de la cadena son dígitos y la cadena no está vacía.

    ```python
    texto1 = "12345"
    texto2 = "123Hola"
    print(texto1.isdigit())  # Output: True
    print(texto2.isdigit())  # Output: False
    ```
*   **`isnumeric()`:** Devuelve `True` si todos los caracteres de la cadena son caracteres numéricos y la cadena no está vacía. Similar a `isdigit()`, pero también incluye caracteres Unicode que representan números.

    ```python
    texto1 = "12345"
    texto2 = "½"
    print(texto1.isnumeric())  # Output: True
    print(texto2.isnumeric())  # Output: True
    ```
*   **`isdecimal()`:** Devuelve `True` si todos los caracteres de la cadena son caracteres decimales y la cadena no está vacía. Más restrictivo que `isnumeric()`.

    ```python
    texto1 = "12345"
    texto2 = "½"
    print(texto1.isdecimal())  # Output: True
    print(texto2.isdecimal())  # Output: False
    ```
*   **`isspace()`:** Devuelve `True` si todos los caracteres de la cadena son espacios en blanco y la cadena no está vacía.

    ```python
    texto1 = "   "
    texto2 = " Hola "
    print(texto1.isspace())  # Output: True
    print(texto2.isspace())  # Output: False
    ```
*   **`islower()`:** Devuelve `True` si todos los caracteres de la cadena están en minúsculas y hay al menos un carácter en minúscula.

    ```python
    texto1 = "hola mundo"
    texto2 = "Hola Mundo"
    print(texto1.islower())  # Output: True
    print(texto2.islower())  # Output: False
    ```
*   **`isupper()`:** Devuelve `True` si todos los caracteres de la cadena están en mayúsculas y hay al menos un carácter en mayúscula.

    ```python
    texto1 = "HOLA MUNDO"
    texto2 = "Hola Mundo"
    print(texto1.isupper())  # Output: True
    print(texto2.isupper())  # Output: False
    ```
*   **`istitle()`:** Devuelve `True` si la cadena está en formato de título (la primera letra de cada palabra en mayúscula).

    ```python
    texto1 = "Hola Mundo"
    texto2 = "hola mundo"
    print(texto1.istitle())  # Output: True
    print(texto2.istitle())  # Output: False
    ```
*   **`isidentifier()`:** Devuelve `True` si la cadena es un identificador válido en Python.

    ```python
    texto1 = "variable_valida"
    texto2 = "1variable"
    print(texto1.isidentifier())  # Output: True
    print(texto2.isidentifier())  # Output: False
    ```
*   **`isprintable()`:** Devuelve `True` si todos los caracteres de la cadena son imprimibles (no son caracteres de control).

    ```python
    texto1 = "Hola Mundo"
    texto2 = "Hola\nMundo"
    print(texto1.isprintable())  # Output: True
    print(texto2.isprintable())  # Output: False
    ```

### METODOS DE FORMATO DE CADENAS

*   **`format(*args, **kwargs)`:** Permite insertar valores en una cadena utilizando marcadores de posición.

    ```python
    nombre = "Juan"
    edad = 30
    texto = "Me llamo {} y tengo {} años.".format(nombre, edad)
    print(texto)  # Output: Me llamo Juan y tengo 30 años.
    ```

    También se pueden usar nombres para los marcadores de posición:

    ```python
    nombre = "Juan"
    edad = 30
    texto = "Me llamo {nombre} y tengo {edad} años.".format(nombre=nombre, edad=edad)
    print(texto)  # Output: Me llamo Juan y tengo 30 años.
    ```

*   **`f-strings` (Python 3.6+):** Una forma más concisa y legible de formatear cadenas.

    ```python
    nombre = "Juan"
    edad = 30
    texto = f"Me llamo {nombre} y tengo {edad} años."
    print(texto)  # Output: Me llamo Juan y tengo 30 años.
    ```

*   **`%`-formatting (estilo antiguo):** Aunque menos común en el código moderno, todavía se puede encontrar.

    ```python
    nombre = "Juan"
    edad = 30
    texto = "Me llamo %s y tengo %d años." % (nombre, edad)
    print(texto)  # Output: Me llamo Juan y tengo 30 años.
    ```

### METODOS DE DIVISION Y UNION DE CADENAS

*   **`split(separador, maxsplit)`:** Divide una cadena en una lista de subcadenas, utilizando un separador específico. El argumento `maxsplit` es opcional y especifica el número máximo de divisiones a realizar.

    ```python
    texto = "Hola Mundo, Hola Python"
    lista = texto.split(", ")
    print(lista)  # Output: ['Hola Mundo', 'Hola Python']
    ```
*   **`rsplit(separador, maxsplit)`:** Similar a `split()`, pero comienza a dividir desde el final de la cadena.

    ```python
    texto = "Hola Mundo, Hola Python"
    lista = texto.rsplit(", ", 1)
    print(lista)  # Output: ['Hola Mundo', 'Hola Python']
    ```
*   **`splitlines(keepends)`:** Divide una cadena en una lista de líneas, utilizando los caracteres de nueva línea como separadores. El argumento `keepends` es opcional y especifica si se deben mantener los caracteres de nueva línea en las líneas resultantes.

    ```python
    texto = "Hola Mundo\nHola Python"
    lista = texto.splitlines()
    print(lista)  # Output: ['Hola Mundo', 'Hola Python']
    ```
*   **`join(iterable)`:** Une una lista de cadenas en una sola cadena, utilizando la cadena como separador.

    ```python
    lista = ["Hola", "Mundo", "Python"]
    texto = " ".join(lista)
    print(texto)  # Output: Hola Mundo Python
    ```

### OTROS METODOS UTILES

*   **`zfill(ancho)`:** Rellena una cadena con ceros a la izquierda hasta alcanzar un ancho específico.

    ```python
    numero = "123"
    numero_rellenado = numero.zfill(5)
    print(numero_rellenado)  # Output: 00123
    ```
*   **`center(ancho, caracter_de_relleno)`:** Centra una cadena dentro de un ancho específico, rellenando los espacios restantes con un carácter de relleno.

    ```python
    texto = "Hola"
    texto_centrado = texto.center(10, "*")
    print(texto_centrado)  # Output: ***Hola***
    ```
*   **`ljust(ancho, caracter_de_relleno)`:** Justifica una cadena a la izquierda dentro de un ancho específico, rellenando los espacios restantes con un carácter de relleno.

    ```python
    texto = "Hola"
    texto_justificado_izquierda = texto.ljust(10, "*")
    print(texto_justificado_izquierda)  # Output: Hola******
    ```
*   **`rjust(ancho, caracter_de_relleno)`:** Justifica una cadena a la derecha dentro de un ancho específico, rellenando los espacios restantes con un carácter de relleno.

    ```python
    texto = "Hola"
    texto_justificado_derecha = texto.rjust(10, "*")
    print(texto_justificado_derecha)  # Output: ******Hola
    ```
*   **`encode(encoding='utf-8', errors='strict')`:** Codifica una cadena en una secuencia de bytes utilizando una codificación específica.

    ```python
    texto = "Hola Mundo"
    bytes_codificados = texto.encode('utf-8')
    print(bytes_codificados)  # Output: b'Hola Mundo'
    ```
*   **`expandtabs(tabsize=8)`:** Reemplaza los caracteres de tabulación en una cadena con un número específico de espacios.

    ```python
    texto = "Hola\tMundo"
    texto_sin_tabs = texto.expandtabs(4)
    print(texto_sin_tabs)  # Output: Hola    Mundo
    ```

### EJEMPLOS DE IMPLEMENTACION

#### EJEMPLO 1: LIMPIEZA DE DATOS

```python
datos = "   Nombre:  Juan Pérez  ,  Edad: 30  "
datos_limpios = datos.strip().replace("Nombre:", "").replace("Edad:", "").split(",")
nombre = datos_limpios[0].strip()
edad = datos_limpios[1].strip()
print(f"Nombre: {nombre}, Edad: {edad}")
```

#### EJEMPLO 2: VALIDACION DE ENTRADA DE USUARIO

```python
def validar_email(email):
    if "@" in email and "." in email:
        return True
    else:
        return False

email = input("Ingrese su email: ")
if validar_email(email):
    print("Email válido")
else:
    print("Email inválido")
```
