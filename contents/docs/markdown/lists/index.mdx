title: Listas
description: Esta sección proporciona una descripción general sobre cómo crear y usar listas en el código fuente de Documentos.
---

Aprenda a crear y representar distintos tipos de listas en el código repetitivo de Documentos. Algunos ejemplos son listas de verificación, listas ordenadas, listas desordenadas y listas anidadas.

## Lista de verificación

- [x] Escribe el comunicado de prensa
- [ ] Actualizar el sitio web
- [ ] Contacta con los medios

```jsx
- [x] Escribe el comunicado de prensa
- [ ] Actualizar el sitio web
- [ ] Contacta con los medios
```

## Lista sencilla

- Item 1
- Item 2
- Item 3

```jsx
- Item 1
- Item 2
- Item 3
```

## Lista de números

1. Research
2. Draft the content
3. Review and edit
4. Publish

```jsx
1. Research
2. Draft the content
3. Review and edit
4. Publish
```

## Lista anidada

- Main Category 1
  - Sub Item 1.1
  - Sub Item 1.2
- Main Category 2
  - Sub Item 2.1
  - Sub Item 2.2

```jsx
- Main Category 1
  - Sub Item 1.1
  - Sub Item 1.2
- Main Category 2
  - Sub Item 2.1
  - Sub Item 2.2
```

Utilice estos ejemplos como base para crear y personalizar listas que se ajusten a las necesidades de su proyecto.
