## Álgebra básica

El área de un círculo `A` se puede calcular utilizando el radio `r` de la siguiente manera:

```math
A = \pi r^2
```

```plaintext
[math]
A = \pi r^2
```

## Fórmula cuadrática

La fórmula cuadrática para resolver una ecuación de la forma es:

```math
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
```

```plaintext
[math]
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
```

## Segunda ley del movimiento de Newton

La segunda ley del movimiento de Newton establece que la fuerza `F` es el producto de la masa `m` y la aceleración `a`:

```math
F = ma
```

## Teorema de Pitágoras

El teorema de Pitágoras relaciona las longitudes de los lados de un triángulo rectángulo:

```math
a^2 + b^2 = c^2
```
```plaintext
[math]
a^2 + b^2 = c^2
```

## Equivalencia masa-energía de Einstein

La famosa ecuación de Einstein relaciona la energía `E`, la masa `m` y la velocidad de la luz `c`:

```math
E = mc^2
```
```plaintext
[math]
E = mc^2
```
