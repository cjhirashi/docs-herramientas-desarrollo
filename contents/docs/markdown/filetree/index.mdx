En esta sección se muestra la estructura de un árbol de archivos mediante el componente `FileTree`. A continuación, se incluye un ejemplo que muestra carpetas y archivos organizados jerárquicamente para la configuración de un proyecto.

<FileTree>
  <Folder name="src" label="Source Code">
    <File name="index.tsx" label="Index File" />
    <Folder name="components" label="Components">
      <File name="button.tsx" label="Button Component" />
      <File name="input.tsx" label="Input Component" />
    </Folder>
    <Folder name="pages" label="Pages">
      <File name="home.tsx" label="Home Page" />
      <File name="about.tsx" label="About Page" />
    </Folder>
  </Folder>
</FileTree>

## Cómo utilizar

Integre el componente `FileTree` en su proyecto para representar visualmente las estructuras de archivos y carpetas. Esto resulta especialmente útil para documentación, tutoriales o para proporcionar a los usuarios una descripción general de la arquitectura de su proyecto.

### JSX Ejemplo de código

A continuación se explica cómo puede definir la estructura del árbol de archivos para utilizarla en su proyecto:

```jsx
<FileTree>
  <Folder name="src" label="Source Code">
    <File name="index.tsx" label="Index File" />
    <Folder name="components" label="Components">
      <File name="button.tsx" label="Button Component" />
      <File name="input.tsx" label="Input Component" />
    </Folder>
    <Folder name="pages" label="Pages">
      <File name="home.tsx" label="Home Page" />
      <File name="about.tsx" label="About Page" />
    </Folder>
  </Folder>
</FileTree>
```

Utilice este código como plantilla para configurar su propia estructura de árbol de archivos y personalizarla según sea necesario.
