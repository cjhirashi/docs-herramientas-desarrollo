## TEMPLATE

Documents es un potente kit de inicio de documentación de Next.js diseñado para simplificar el proceso de creación de documentación de productos, manuales técnicos y guías comerciales integrales y de alta calidad. Desarrollado con React, Tailwind CSS y TypeScript, este kit de inicio proporciona la base para crear la documentación de su proyecto y su base de conocimientos.

### STACK TECNOLOGICO

- **Framework**: Next.js
- **Caso de uso**: Documentación
- **CSS**: Tailwind

### CARACTERISTICAS

- **Creación de contenido**
  - ***Integración MDX***: escriba documentos con componentes Markdown y JSX.
  - ***Componentes personalizados***: reutilice componentes React en sus documentos.
  - ***Diagramas de Mermaid.js***: crea diagramas de flujo y diagramas.
  - ***Matemáticas y tablas***: incluye tablas y fórmulas matemáticas LaTeX.

- **Navegación y legibilidad**
  - ***Navegación multinivel***: páginas anidadas para crear jerarquías.
  - ***Paginación de contenido***: transiciones fluidas entre páginas.
  - ***Tabla de contenidos dinámica***: tabla de contenidos generada automáticamente.
  - ***Selector de código***: alterne y copie fragmentos de código fácilmente.

- **Características del código**
  - ***Resaltado de sintaxis***: Resaltado según tema.
  - ***Bloques de código mejorados***: Resaltado de líneas y títulos de código.

- **Temas y experiencia de usuario**
  - ***Modos claro/oscuro***: Cambio automático de tema.
  - ***Optimización SEO***: Metadatos generados automáticamente para una mejor clasificación.
    - Gestión de metaetiquetas (título, descripción, URL, uso compartido en redes sociales).

- **Búsqueda y mejoras futuras**
  - ***Búsqueda avanzada***: Búsqueda aproximada con resaltado de términos.
  - ***Base de conocimiento de IA***: Futuras herramientas de documentación impulsadas por IA.

## CLONAR REPOSITORIO 

<Step>
  <StepItem title="Clonar repositorio">

    Clona el repositorio de GitHub y accede a la nueva carpeta del proyecto
   
    ```bash
    git clone https://github.com/rubixvi/rubix-documents.git
    cd rubix-documents
    ```
  </StepItem>

  <StepItem title="Conectar con nuevo repositorio">

    1. Crea un nuevo repositorio en [GitHub](https://github.com/)
    2. Verificar qué origen tiene asignado el repositorio del proyecto
        ```bash
        git remote -v
        ```
        Esto mostrará una respuesta como la siguiente:
        ```bash
        origin  <URL REPO ASIGNADO> (fetch)
        origin  <URL REPO ASIGNADO> (push)
        ```
    2. Ejectura el comando de reasignación de Origen del proyecto con el siguiente comando:
        ```bash
        git remote set-url origin <NUEVA_URL>
        ```


  </StepItem>

  <StepItem title="Instalar dependencias">

    Instala todas las dependencias del proyecto
    
    ```bash
    npm install
    ```

  </StepItem>

  <StepItem title="Correr el servidor">

    - **Ejecutar en modo desarrollo**:
    
      ```bash
      npm run dev
      ```

    - **Ejecutar en modo producción**:
      1. Construir aplicación:
        ```bash
        npm run build
        ```
      2. Ejecutar en modo producción:
        Construir aplicación:
        ```bash
        npm run start
        ```

  </StepItem>
</Step>
