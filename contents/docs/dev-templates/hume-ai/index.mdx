## CLONAR REPOSITORIO

<Step>
  <StepItem title="Clonar repositorio">

  Clona el repositorio de GitHub y accede a la nueva carpeta del proyecto
   
    ```bash
    git clone https://github.com/HumeAI/hume-evi-next-js-starter.git
    cd hume-evi-next-js-starter
  ```

  </StepItem>

  <StepItem title="Conectar con nuevo repositorio de GitHub">
    
    1. Crea un nuevo repositorio en [GitHub](https://github.com/)
    2. Verificar qué origen tiene asignado el repositorio del proyecto clonado
        ```bash
        git remote -v
        ```
    2. Ejectura el comando de reasignación de Origen del proyecto con el siguiente comando:
        ```bash
        git remote set-url origin [URL de nuevo repositorio]
        ```
  </StepItem>

  <StepItem title="Instalar dependencias">
    
    Instala todas las dependencias del proyecto
    
    ```bash
    pnpm install
    ```
  </StepItem>
  <StepItem title="Variables de entorno">
    
    Proporcione las variables de entorno necesarias. Para obtener su clave API y su clave secreta de cliente, inicie sesión en el portal y visite la página de claves API.
    
    ```bash
    HUME_API_KEY=<API KEY>
    HUME_SECRET_KEY=<SECRETE KEY>
    NEXT_PUBLIC_HUME_CONFIG_ID=<HUME CONFIG ID>
    ```
  </StepItem>
  <StepItem title="Correr el servidor">

    - **Ejecutar en modo desarrollo**:
    
      ```bash
      pnpm run dev
      ```

    - **Ejecutar en modo producción**:
      1. Construir aplicación:
        ```bash
        pnpm run build
        ```
      2. Ejecutar en modo producción:
        Construir aplicación:
        ```bash
        pnpm run start
        ```

  </StepItem>
</Step>

## HERRAMIENTAS 

### HUME AI

[https://platform.hume.ai/](https://platform.hume.ai/)

Hume AI crea modelos de IA que permiten que la tecnología se comunique con empatía y aprenda a hacer felices a las personas.

Gran parte de la comunicación humana (en persona, por texto, audio o video) está determinada por la expresión emocional. Estas señales nos permiten prestar atención al bienestar de los demás. Nuestra plataforma proporciona las API necesarias para garantizar que la tecnología también esté guiada por la empatía y la búsqueda del bienestar humano.

#### INTERFAZ DE VOZ EMPATICA 
La interfaz de voz empática (EVI) de Hume es la primera IA de voz emocionalmente inteligente del mundo. Es la única API que mide modulaciones vocales matizadas y responde a ellas utilizando un modelo de lenguaje empático amplio (LLM), que guía la generación del lenguaje y el habla. Entrenado en millones de interacciones humanas, nuestro LLM une el modelado del lenguaje y la conversión de texto a voz con mejor ecualización, prosodia, detección de final de turno, interrumpibilidad y alineación.

#### CONVERSION DE TEXTO A VOZ 
Octave TTS, el primer sistema de conversión de texto a voz basado en inteligencia LLM. A diferencia de los TTS convencionales que simplemente "lee" palabras, Octave es un "modelo de habla y lenguaje" que entiende lo que significan las palabras en contexto, lo que permite alcanzar un nuevo nivel de expresividad y matices.

#### MEDICION DE LA EXPRESION 
Los modelos de medición de la expresión de vanguardia de Hume para la voz, el rostro y el lenguaje se basan en más de 10 años de investigación y avances en la teoría del espacio semántico, iniciada por Alan Cowen. Nuestros modelos de medición de la expresión pueden capturar cientos de dimensiones de la expresión humana en audio, video e imágenes.
