## DESCRIPCION GENERAL 

Este proyecto presenta una implementación de muestra de la interfaz de voz empática de Hume con el SDK de React de Hume. Aquí, tenemos una EVI simple que utiliza el enrutador de aplicaciones Next.js.

### STACK TECNOLOGICO

- **Framework**: Next.js
- **Caso de uso**: AI
- **CSS**: Tailwind

## IMPLEMENTACION DEL PROYECTO 

Haga clic en el botón a continuación para implementar este proyecto de ejemplo con Vercel:

A continuación, se muestran los pasos para completar la implementación:

1. Clona el repositorio de GitHub

  ```bash
  https://github.com/HumeAI/hume-evi-next-js-starter.git
  ```

2. Instala las dependencias

```bash
pnpm install
```

3. Proporcione las variables de entorno necesarias. Para obtener su clave API y su clave secreta de cliente, inicie sesión en el portal y visite la página de claves API.

```bash
HUME_API_KEY=<API KEY>
HUME_SECRET_KEY=<SECRETE KEY>
NEXT_PUBLIC_HUME_CONFIG_ID=<HUME CONFIG ID>
```
