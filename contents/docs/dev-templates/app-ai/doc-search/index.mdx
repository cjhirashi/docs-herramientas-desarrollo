## DESCRIPCION GENERAL 

Este iniciador toma todos los archivos .mdx en el directorio de páginas y los procesa para usarlos como contexto personalizado dentro de las indicaciones de Completar texto de OpenAI.

### STACK TECNOLOGICO

- **Framework**: Next.js
- **Caso de uso**: AI
- **CSS**: Tailwind
* **Database**: Supabase
